# Здесь используются
- **Цветовая палитра:** [Gruvbox Material](https://github.com/sainnhe/gruvbox-material)

- **WM:** niri

- **Music:** rmpc + mpd

- **Shell:** zsh

- **Terminal:** kitty

- **Utilities:** bat (cat, только получше)

- **Info:** fastfetch + btop

- **Launcher:** fuzzel

- **Bar:** waybar

- **File Manager:** yazi (ranger, только на Rust)

- **System:** swaync + swaylock + swayidle

- **Telegram:** кастомная тема в цвет системы
